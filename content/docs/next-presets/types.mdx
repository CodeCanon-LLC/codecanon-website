All types are exported directly from `@codecanon/next-presets`.

```tsx
import type {
  PresetKeys,
  ColorScheme,
  Theme,
  ThemeProviderProps,
  PresetProviderProps,
} from "@codecanon/next-presets";
```

## ColorScheme

The resolved color mode — never includes `"system"`.

```ts
type ColorScheme = "light" | "dark";
```

---

## Theme

All theme values a user can select, including the `"system"` option.

```ts
type Theme = "light" | "dark" | "system";
```

---

## ThemeProviderProps

Props accepted by `<ThemeProvider>`.

```ts
type ThemeProviderProps = {
  children: React.ReactNode;
  defaultTheme?: Theme;
  themeKey?: string;
  // + all next-themes ThemeProviderProps
};
```

---

## PresetProviderProps

Props accepted by `<PresetProvider>`.

```ts
type PresetProviderProps = {
  children: React.ReactNode;
  defaultPreset?: PresetKeys;
  presetKey?: string;
};
```

---

## PresetState

Returned by `usePreset()`.

```ts
type PresetState = {
  preset?: PresetKeys;
  setPreset: (preset: PresetKeys) => void;
  resetPreset: () => void;
};
```

---

## PresetPickerState

Returned by `usePresetPicker()`.

```ts
type PresetPickerState = {
  open: boolean;
  setOpen: (value: boolean) => void;
  toggleOpen: () => void;
};
```

---

## PRESETS

The full preset registry — a readonly array of `[id, label]` tuples.

```ts
const PRESETS: readonly [PresetKeys, string][];
```

---

## DEFAULT_PRESET

The preset used when no stored value exists and no `defaultPreset` prop is provided.

```ts
const DEFAULT_PRESET: PresetKeys; // "nuteral"
```
