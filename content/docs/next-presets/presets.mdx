Each preset defines a complete set of CSS variables for **both light and dark mode** — covering background, foreground, primary, secondary, muted, accent, destructive, border, input, ring, sidebar colors, and five chart colors.

## How presets work

When a preset is active, `PresetProvider` sets `data-preset="<id>"` on `document.documentElement`. The imported stylesheet uses these selector groups:

```css
/* Light mode */
[data-preset="violet-bloom"],
[data-preset="violet-bloom"].dark .scheme-light,
:root[data-preset="violet-bloom"] {
  --primary: oklch(0.6 0.25 290);
  --background: oklch(0.98 0.01 290);
  /* ... full light mode token set */
}

/* Dark mode */
[data-preset="violet-bloom"] .dark,
[data-preset="violet-bloom"] .scheme-dark,
[data-preset="violet-bloom"].dark {
  --primary: oklch(0.75 0.22 290);
  --background: oklch(0.1 0.02 290);
  /* ... full dark mode token set */
}
```

This means **presets and dark/light mode are fully independent** — every preset works in both modes. The `.scheme-light` / `.scheme-dark` selectors allow scoping light or dark mode to a specific subtree.

## Using a preset

```tsx
import { usePreset } from "@codecanon/next-presets";

const { setPreset } = usePreset();

setPreset("violet-bloom");
```

## Listing all presets

```tsx
import { PRESETS, DEFAULT_PRESET } from "@codecanon/next-presets";

// Array of [id, label] tuples
PRESETS.forEach(([key, label]) => {
  console.log(key, label);
});

// Default preset ID
console.log(DEFAULT_PRESET); // "nuteral"
```

## Custom presets

You can extend the preset system with your own CSS. Define the preset's variables using `[data-preset="my-preset"]` selectors and add them to your global stylesheet after the library styles:

```css
/* globals.css — after @import "@codecanon/next-presets/styles.css" */

/* Light mode */
[data-preset="my-brand"],
[data-preset="my-brand"].dark .scheme-light,
:root[data-preset="my-brand"] {
  --background: oklch(0.99 0.005 220);
  --foreground: oklch(0.1 0 0);
  --primary: oklch(0.45 0.2 220);
  --primary-foreground: oklch(0.98 0 0);
  --secondary: oklch(0.93 0.02 220);
  --secondary-foreground: oklch(0.2 0 0);
  --muted: oklch(0.95 0.01 220);
  --muted-foreground: oklch(0.5 0 0);
  --accent: oklch(0.9 0.04 220);
  --accent-foreground: oklch(0.15 0 0);
  --destructive: oklch(0.577 0.245 27.325);
  --border: oklch(0.88 0.01 220);
  --input: oklch(0.88 0.01 220);
  --ring: oklch(0.55 0.18 220);
  --radius: 0.5rem;
}

/* Dark mode */
[data-preset="my-brand"] .dark,
[data-preset="my-brand"] .scheme-dark,
[data-preset="my-brand"].dark {
  --background: oklch(0.1 0.005 220);
  --foreground: oklch(0.97 0 0);
  --primary: oklch(0.65 0.18 220);
  /* ... dark mode values */
}
```

Then activate it with `setPreset`:

```tsx
const { setPreset } = usePreset();
setPreset("my-brand");
```
